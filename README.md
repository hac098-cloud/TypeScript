# Event Analytics Dashboard

This is a simple TypeScript project that collects product events and displays basic analytics in a dashboard.

## Features

- Add new events
- View total event count
- View today's event count
- View active users
- View purchase events
- View events by type
- View events over time
- View recent events in a table
- Save events using localStorage

## Tech Stack

- React
- TypeScript
- Vite
- Recharts

## How to Run

```bash
npm install
npm run dev
